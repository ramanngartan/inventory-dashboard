# Inventory Dashboard

A full-stack Inventory Management Dashboard built with the MERN stack.

This project is being developed as a portfolio application to demonstrate modern full-stack development practices, including authentication, CRUD operations, analytics, image uploads, and deployment.

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features (Planned)

- User Authentication (JWT)
- Dashboard Overview
- Product Management
- Category Management
- Search & Filtering
- Pagination
- Charts & Analytics
- Image Uploads
- Responsive UI
- Deployment

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ramanngartan/inventory-dashboard.git
```

### Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run the project

**Backend**

```bash
npm run dev
```

**Frontend**

```bash
npm run dev
```
